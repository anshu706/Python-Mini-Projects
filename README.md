# C Mini Projects

A collection of **8 beginner-level C console applications** covering banking, games, inventory management, and reference tools.

🌐 **[View the Showcase Website →](https://anshu706.github.io/C-Mini-Projects/)**

---

## Projects

| # | File | Description |
|---|------|-------------|
| 1 | [`atm.c`](atm.c) | ATM Banking System – card auth, PIN verification, withdrawal & balance check |
| 2 | [`billing.c`](billing.c) | Supermarket Billing System – itemised receipt across 3 product categories |
| 3 | [`casino.c`](casino.c) | Casino Number Game – deposit, bet, guess 1–10, win 10× your stake |
| 4 | [`guess_game.c`](guess_game.c) | Guess the Number – classic 3-attempt guessing game |
| 5 | [`lib_manage.c`](lib_manage.c) | Library Management System – browse, search & view books by department |
| 6 | [`mod_periodic_table.c`](mod_periodic_table.c) | Periodic Table Viewer – look up elements by atomic number |
| 7 | [`phone_book.c`](phone_book.c) | Phone Book – add & view contacts with full details |
| 8 | [`quiz_game.c`](quiz_game.c) | Quiz Game – 5-question CS history MCQ with live scoring |

## Getting Started

### Prerequisites

- GCC or any C99-compatible compiler

### Compile & Run

```bash
# Example: compile and run the ATM simulation
gcc atm.c -o atm && ./atm

# Example: compile and run the Quiz Game
gcc quiz_game.c -o quiz && ./quiz
```

## Website

The `docs/` folder contains a static showcase website built with plain HTML, CSS and JavaScript.

To run it locally open `docs/index.html` in a browser, or serve it with any static file server:

```bash
npx serve docs
```
