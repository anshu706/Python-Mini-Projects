About the Collection

These mini-projects are beginner-friendly C programs built to practise core language constructs: I/O with scanf/printf, control-flow with if-else and switch, loops, functions and simple data modelling using structs and arrays.

Each program runs entirely in the terminal and demonstrates a different real-world use-case — from banking and retail to games and reference tools.

Clone on GitHub ↗

Concepts Covered

  • Variables & Data Types
  • Control Flow (if/else, switch)
  • Loops (for, while)
  • Functions & Recursion
  • Arrays & Strings
  • Structs
  • User Input / Output
  • Random Number Generation

Game

Casino Number Game

casino.c

A two-round gambling simulation. The player deposits a starting balance, places a bet and guesses a number between 1 and 10. Guessing correctly earns 10× the wager; guessing wrong deducts the bet. A second round is offered if the player still has funds.

Game Flow

  1. Enter name and deposit amount.
  2. Place a bet (must be ≤ balance).
  3. Guess a number from 1 to 10.
  4. Win 10× or lose the bet amount.
  5. Choose to play another round or quit.
